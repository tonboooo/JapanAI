# Lyon Display & Lyon Text フォントについて

このフォルダには以下のフォントファイルを配置してください：

## Lyon Display Web（見出し用）
- LyonDisplay-Medium.woff2 - ウェイト500
- LyonDisplay-Bold.woff2 - ウェイト700
- LyonDisplay-Black.woff2 - ウェイト900

## Lyon Text Web（本文用）
- LyonText-Regular.woff2 - ウェイト400
- LyonText-Semibold.woff2 - ウェイト600

## 注意事項
- Lyon DisplayおよびLyon TextはCommercial Type社が販売するライセンスフォントです
- 使用にはライセンスの購入が必要です
- 購入後にwoff2ファイルをこのフォルダに配置してください
- https://commercialtype.com/catalog/lyon からライセンスを取得できます

## 一時的な代替方法
Lyon Displayフォントが購入されるまでは、以下の無料フォントを代替として使用できます：
- 見出し用: Playfair Display（Google Fonts）
- 本文用: Lora（Google Fonts）

これらを使用する場合は globals.css のフォント定義を適宜修正してください。
